# 📖 Overview

The **HR Attrition Analytics Dashboard** analyzes the **IBM HR Analytics Employee Attrition & Performance** dataset containing **1,470 employee records** and **35 workforce-related features** to identify the primary factors influencing employee turnover.

Using **Python for Exploratory Data Analysis (EDA)** and **Power BI for interactive dashboard development**, the project uncovers workforce trends, attrition drivers, demographic insights, and department-level performance indicators to support strategic HR planning.

# 📈 Key Insights

This analysis revealed several important workforce trends, including:

- High overtime employees exhibited significantly higher attrition.
- Employees with lower monthly income showed increased turnover rates.
- Early-career employees experienced higher attrition than long-tenured staff.
- Certain job roles and departments contributed disproportionately to overall attrition.
- A high-risk employee segment was identified with an **attrition rate of approximately 68%**, compared to the **overall organizational attrition rate of around 16%**.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/hr-attrition-dashboard.git
```

Navigate into the project

```bash
cd hr-attrition-dashboard
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```
